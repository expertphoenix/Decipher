The Rules

  1. This is a fun, challenging puzzle, inspired by McGovern's Zahada
  2. Remove any spaces from answers and only answer in lowercase
  3. Each puzzle has only one correct answer
  4. Almost always, it will be obvious when you have the right answer so don't spend ages guessing
  5. Searching is recommended if you are stuck with some complex words
  6. Upon finding the answer, replace the part of the URL before .html with it, like this: https://aurorasylvani.github.io/Decipher/insertyouranswerhere.html
  7. Restricted act: Cheating (because it'll ruin the fun and you should be ashamed of yourself)
  8. Every level number is always in the top left corner

Good luck!
